How to change color theme

Changing your template Theme color is simple. You need to modify 1 line CSS code.

Got to your /asset/css/ folder you should be able to see all of the available skins (skin-1.css skin-2.css...etc). Our Default color theme is skin-1.css.

Now open up the style.css file and find the Import section. You will see @import url("skin-1.css");. and replace skin name with your choice like@import url("skin-2.css");

Customize Slider

The jQuery Cycle Plugin is a slideshow plugin that supports many different types of transition effects. Easy to cutomize and simple.

jquery cycle demos and documentation

If you want to change the slider's speed transition, open up the file home.js and change the following code

Home page slider version 1 

$('.slider-v1').cycle({
        //Specify options
        fx: 'scrollHorz', //Name of transition effect 
        slides: '.slider-item',
        timeout: 5000, // set time for next slide 
        speed: 1200,
